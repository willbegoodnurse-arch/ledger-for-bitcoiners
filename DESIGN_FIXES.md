My Ledger — 디자인 수정 지시
실제 구현 화면을 검토한 결과, 아래 4가지를 수정한다. 우선순위 순.

1. 🔴 [홈] 번개가 안 보인다 — 가장 중요
문제: 번개 애니메이션이 끝나면 완전히 투명(opacity 0)으로 사라져서, 주기 중 대부분 우측 상단이 텅 비어 보인다. 핵심 컨셉인데 안 보인다.

수정:

번쩍인 뒤 완전히 사라지지 말고 은은한 잔상으로 남게 한다.
번개 wrapper 기본 opacity: 0 → 0.16
strike 키프레임 끝부분을 0이 아니라 잔상으로 착지:
55% { opacity: 0.45; filter: brightness(1.1); }
80% { opacity: 0.24; filter: brightness(1.0); }
100% { opacity: 0.16; filter: brightness(0.95); }
진입 직후 빠르게 한 번: 초기 strike 트리거 700ms → 300ms
자동 발사 주기 5s → 3.5s
우측 상단에 더 확실히 걸치게: size 0.7→0.82, offsetX 4→1, offsetY -4→-3
z-index 최하단(텍스트 뒤) 유지 — 슬로건/숫자를 절대 가리지 않는다.
2. 🔴 [자산] "적립 추이 (누적 SATS)" 차트가 텅 비어 있다
문제: 거래가 1건이라 누적 그래프가 안 그려져 빈 박스 → 깨진 것처럼 보인다.

수정 (택1 또는 병행):

1~2건이어도 점/영역이 보이게 렌더 (단일 포인트면 도트라도 찍기)
정말 비면 빈 상태 안내: "아직 적립 내역이 없어요. 첫 DCA를 기록해보세요."
데모 품질을 위해 DCA Buy 더미 거래를 3~5건 더 추가해도 좋다.
3. 🟡 [거래 입력] 폼 여백 과다 + 제출 버튼이 붕 떠 있다
수정:

필드(구분/금액/카테고리/제목/메모/날짜) 간 세로 간격 축소
제출 버튼을 폼 카드 바로 아래로 붙이고 하단 여백 제거
모바일 한 화면(약 820px 높이)에 자연스럽게 차도록 밀도 조정
4. 🟡 [통계] 인사이트 문구가 줄 끝에서 단어째 쪼개진다
문제: "...155,285 sats 예 / 요." 처럼 단어/숫자가 줄바꿈에서 분리.

수정:

인사이트 텍스트에 word-break: keep-all;
숫자+단위(155,285 sats)는 white-space: nowrap으로 묶기
text-wrap: pretty; 권장
